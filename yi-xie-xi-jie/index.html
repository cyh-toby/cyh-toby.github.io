一些细节

2020-08-17 16:22 1 min read # 记录

  • BFS 在边权为 1 的图中找最短路可视情况加入 if (now == goal) return f[now]; 这种语句，但 SPFA 中绝不可以脑残加入这玩意。

  • 'std::sort' 及类似函数中，编写 'bool cmp()' 或重载须满足在此规则下，不存在 'x < y' 且 'y < x' ，如

    inline bool cmp(int x, int y) {
    return x <= y;
}

就不是合法规则。详见 https://www.cnblogs.com/hliu17/p/7399965.html

  • 不要在 #define 的时候手乱瞎写，把 (i<<1) 写成 (1<<i) 等，调死都发现不了。

  • 尽量减少宏的使用，不好调错。
