学习笔记：差分约束

2020-01-16 22:15 6 min read # 学习笔记

概念

差分约束：如果一个系统由nnn个变量和mmm个约束条件组成，形成mmm个形如 ai−aj≤kai-aj≤kai−aj≤k 的不等式(i,j∈[1,n],k(i,j∈[1,n],k(i,j∈[1,n],k为常数),),),则称其为差分约束系统(system−of−differenceconstraints)(system-of-difference constraints)(system−of−differenceconstraints)。亦即，差分约束系统是求解关于一组变量的特殊不等式组的方法。

引理

在谈及如何实现差分约束系统之前，先看看图论-最短路中的SPFASPFASPFA算法。此算法类似于BFSBFSBFS，以O(n)O(n)O(n) ~ O(nm)O(nm)O(nm)的时间复杂度求出图中的单源最短路。

而对于SPFASPFASPFA而言，稍加修改即可判断负权/正权回路：

if(dis[y]<dis[x]+w[x,y]) {
	dis[y] = dis[x]+w[x,y];
    	cnt[y] = cnt[x]+1;
    	if(cnt[y]>=n) return false;//发现负权回路
  ...

详见以后将会给出的学习笔记：图论-最短路。

分析

回到差分约束的定义，我们可以把题目条件化成形如这样的式子：

x[i]−x[j]≤c[k]x[i]-x[j] ≤ c[k]x[i]−x[j]≤c[k]

<=>

x[i]≤x[j]+c[k]x[i] ≤ x[j]+c[k]x[i]≤x[j]+c[k] ①

而在SPFASPFASPFA求最短路中，结束程序时，对于任意∀i,j∈G∀i,j ∈ G∀i,j∈G，恒有：

dis[i]≤dis[j]+w[j,i]dis[i] ≤ dis[j] + w[j,i]dis[i]≤dis[j]+w[j,i] ②

观察①、②两式，形式极为类似！

这启发我们拿到差分约束条件后，构建一张有向图GGG：

  • GGG中的节点编号代表待解不等式组中的未知数下标

  • dis[i]dis[i]dis[i]表示原不等式组中的x[i]x[i]x[i]的值

  • 对于每个不等关系式x[i]−x[j]≤c[k]x[i]-x[j] ≤ c[k]x[i]−x[j]≤c[k]，建立一条由jjj指向iii、权值为c[k]c[k]c[k]的有向边

这样一来，是不是跑一遍SPFASPFASPFA就可以求出合法解了？

但还有两个漏洞！！

漏洞1-不等式组无解

我们知道，如果存在负权回路，SPFASPFASPFA是停不下来的，那如何求解？其实图GGG存在负权回路等价于原不等式组无解。
考虑某套不等式组（可能只是原不等式组的一部分）：

x1 <= x2 + c1;
x2 <= x3 + c2;
	...
xn-1 <= xn + cn-1;
xn <= x1 + cn;

把上述不等式组相加，可得：

0 <= c1 + c2 + ... + cn//应恒成立

这意味着，如果c1+c2+...+cn≤0c1+c2+...+cn ≤ 0c1+c2+...+cn≤0 ， 那么这一部分不等式组无解，进而导致整个差分约束无解。

而这正对应着图GGG中存在某个环，其边权和小于000——即负权回路！

证明了负权回路对应无解；而若不存在负权回路，SPFASPFASPFA一定可以求出一组合法解，所以证明了存在负权回路和不等式组无解的等价性。

漏洞2-全图遍历的完整性

我们知道，只有当遍历完全图所有边和点，才能使求得的解满足所有不等关系（即整个差分约束）。而随便选择一个节点开始遍历，是不能确保的。

那如何确保遍历完所有的边和点呢？

考虑像判负权回路一样，建立虚拟000节点，并将其以边权000指向所有的节点。此时我们从000节点开始遍历图GGG，就可以确保遍历到所有点和边了。

此外还有一个重要的隐藏结论！

观察调用完SPFASPFASPFA(求最短路)后关于节点000的不等关系：

∀i∈G∀i∈G∀i∈G，有dis[i]≤dis[0]+w[0,i]dis[i] ≤ dis[0] + w[0,i]dis[i]≤dis[0]+w[0,i]

即： dis[i]≤dis[0]dis[i] ≤ dis[0]dis[i]≤dis[0]

这意味着，我们可以通过事先修改dis[0]dis[0]dis[0]的初始值，限制整个差分约束所有解的范围（比如题目限制极值等）

但有个小细节：SPFASPFASPFA算法中，所有不等号方向一致，所以：

  • 如果题目要满足∀xi≤S∀xi≤S∀xi≤S，则令dis[0]=Sdis[0] = Sdis[0]=S，并求最短路（即与上述分析不变）
  • 如果题目要满足∀xi≥S∀xi≥S∀xi≥S，则令dis[0]=Sdis[0] = Sdis[0]=S。由于SPFASPFASPFA中不等号方向不可改变，所以只能把原不等式组化成 x[j]≥x[i]−c[k]x[j]≥x[i]-c[k]x[j]≥x[i]−c[k] ，经过与上述类似的分析，这相当于建立边权为−c[k]-c[k]−c[k] 的有向边 i−>ji->ji−>j，
    然后求最长路

小结

对于差分约束系统，我们的处理步骤如下：

  1. 分析题意
  2. 建图
  3. 建立虚拟源点，判正/负权回路
  4. 为虚拟点赋点权
  5. 求最短路，即为所求

值得注意的是，设计此知识点的题目肯定 不会直接 告诉你是差分约束，往往需要 分析题意 乃至结合二分、前缀和等算法。
