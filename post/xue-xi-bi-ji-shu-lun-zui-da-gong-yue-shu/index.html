学习笔记：数论-最大公约数

2020-03-15 20:55 2 min read # 学习笔记

相关概念/定义/性质

公约数就会公有的约数。

显然最大公约数(gcd)就是最大的那个。

最小公倍数(lcm)类似。

有公式：

x×y=gcd⁡(x,y)×lcm(x,y)x\times y = \gcd(x,y)\times \text{lcm}(x,y) x×y=gcd(x,y)×lcm(x,y)

证明比较显然，分别把 gcd 和 lcm 设出来然后倒一倒，详细过程略。

求解最大公约数

引理：gcd⁡(a  ,b)=gcd⁡(b  ,  a mod b)\gcd(a\;,b)=\gcd(b\;,\;a\bmod b)gcd(a,b)=gcd(b,amodb)

引理证明：

  • 记 a=kb+ra=kb+ra=kb+r ，设 a,ba,ba,b 的某一个公因数为 ttt ，即 a≡0(modt)  ,  b≡0(modt)a\equiv0\pmod t\;,\;b\equiv0\pmod ta≡0(modt),b≡0(modt)，而 r=a−kbr=a-kbr=a−kb，故 r≡0(modt)r\equiv0\pmod tr≡0(modt) 。所以 ttt 为 r,br, br,b 公因数，即 ttt 为 b,a mod bb,a\bmod bb,amodb 的公因数。

  • 同理可证 b,a mod bb, a\bmod bb,amodb 的公因数亦为 a,ba, ba,b 的公因数。

所以，它们所有因数等价。因此，最大公因数也等价。即引理命题成立。

有了这个引理，就可以递归求解 gcd⁡\gcdgcd 了。

int gcd(int a, int b) {
	if (b == 0)
		return a;
	return a % b;
}
