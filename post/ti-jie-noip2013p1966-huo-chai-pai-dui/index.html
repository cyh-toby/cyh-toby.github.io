题解 [NOIp2013]/P1966 【火柴排队】

2020-03-08 20:09 3 min read # 题解

题意

原题：P1966 火柴排队

  • 给定两个序列 aaa ，bbb ，同一序列无重，每次可以交换某一序列的相邻两位置，最终使 ∑(ai−bi)2\sum(a_i-b_i)^2∑(ai​−bi​)2 最小。

  • 求最小操作次数。

分析

不难得知，要满足上式最小，应该小的和小的配对，大的和大的配对。

具体而言，就是让 aaa 和 bbb 中的第 kkk 大数在同一位置 iii 即可

由【物理知识：运动是相对的】知：任意操作在 aaa 上做和在 bbb 上做是等价的。

那么为了简化问题，统一以 aaa 为范本，在 bbb 上操作。

所以，我们需要预先处理 aaa 中第 kkk 大的数的位置，随后让 bbb 中的相同地位的数与 aaa 数组中的位置匹配起来。

具体而言就是把 aaa 中第 kkk 大数的位置给 bbb 中第 kkk 大的数记下来，再求出这个位置序列的逆序对个数，即是答案。

求逆序对可以用树状数组或者归并排序

具体参考代码

源码

/**********************************

Problem: [NOIpTG2013]火柴排队 

State: Accepted

From: [NOIpTG2013]

Algorithm: 归并排序, 逆序对 

Author: cyh_toby

last updated on 2020.3.8

**********************************/
#include <cstdio>
#include <algorithm>
using namespace std;
const int N = 100005, mod = 99999997;
int n, a[N], b[N];
int ans = 0;
struct h{
	int v, id;
} c[N], d[N];
void merge(int l, int mid, int r)//归并排序求逆序对个数 
{
	if (l >= r) return; 
	merge(l, (l+mid)>>1, mid);
	merge(mid+1, (mid+1+r)>>1, r);
	int i = l, j = mid+1;
	for (int k = l; k <= r; k++) 
		if (j > r || (i <= mid && a[i] < a[j]))
			b[k] = a[i++];                                        
		else
			b[k] = a[j++], ans = (ans + mid - i + 1) % mod;
	for (int k = l; k <= r; k++)
		a[k] = b[k];
	return;
}
inline bool cmp(h x, h y)
{
	return x.v < y.v;
}
int main()
{
	scanf("%d", &n);
	for (int i = 1; i <= n; i++) {
		scanf("%d", &c[i].v);
		c[i].id = i;
	}
	sort(c+1, c+1+n, cmp);	
	for (int i = 1; i <= n; i++) {
		scanf("%d", &d[i].v);
		d[i].id = i;
	}
	sort(d+1, d+1+n, cmp);
	for (int i = 1; i <= n; i++) {
		a[d[i].id] = c[i].id;
	}	
	merge(1, (1+n)>>1, n);
	printf("%d", ans);
	return 0;
}
