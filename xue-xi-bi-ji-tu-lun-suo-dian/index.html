学习笔记：图论-缩点

2020-03-03 09:27 9 min read # 学习笔记

强连通分量(scc)

  • 针对有向图定义(无向图就等价于连通块了)

  • 通俗来讲，在有向图某个点集中，如果每个点都可以到达这个点集中其他所有点，则此点集为一个强连通分量。

  • 一般，强连通分量允许只包含一个点

拓扑排序(topsort)

  • 针对有向图定义

  • 把入度为 000 的点丢到序列最前端

  • 通过某种排序，使得： 从某个入度为 000 的点进入图后，在同一条路径上，排序中靠前的点总比靠后的点先访问到。

  • 可以转化为课程修学安排理解(广义讲就是AOE网)

    如图：

    此处找到一种合理的学习顺序，就是拓扑排序

  • 只有 有向无环图 (DAG) 可以实现拓扑排序

作用

  • 如果我们可以把图中的所有强联通分量找到，并把所有强连通分量中的点都分别打包压缩成一个大点，再由这些大点构成一张新图，那么新图就是一个 有向无环图
    即 DAG ，那么就可以实现拓扑排序。

    值得注意的是，我们需要寻找最优的强连通分量。例如：

    只有这样，缩点后的图才是 DAG。

强连通分量

求强连通分量一般来讲有两种算法

tarjan 和 kosaraju\text{tarjan 和 kosaraju} tarjan 和 kosaraju

tarjan 时间复杂度是O(n+m)O(n+m)O(n+m)的，十分高效。相比之下，kosaraju 复杂度略高(常数)，适用范围更局限，但更好理解。

显然 OI 不能追求好理解，一般我们会采用高效的 tarjan 算法(也不是很难，就是不好讲)

tarjan 算法的核心是两个数组和一个栈

  • 栈：每访问到一个点，入栈；被归到某个强连通分量后，出栈

  • dfn[u]dfn[u]dfn[u]：uuu 号点的时间戳，即第几个被访问的节点(类似于 dfs 序)

  • low[u]low[u]low[u]：从 uuu 节点出发，可以访问到的 dfndfndfn 最小且处在栈中的节点的 dfndfndfn 值。初始值与自己 dfndfndfn 相等。

  • 显然访问了所有子节点后，根据定义，如果 low[u]==dfn[u]low[u] == dfn[u]low[u]==dfn[u] ，说明 uuu 无法到达比自己更早的节点了。而此时栈中所保留的点能到达最早至少是 uuu (否则它们就在被访问时出栈，归入另一个强连通分量，即经历了此步骤)。那么从栈顶到 uuu 的所有节点都可以切仅可以最早到达 uuu ，那么它们在一个强连通分量中。

  • 接着，考虑如何计算 lowlowlow 。假设先处在 uuu 号点，遍历到一条边 (u, v)(u,\,v)(u,v)

    • uuu 是 vvv 的父亲。由定义，显然

    low[u]=min(low[u],  low[v])low[u] = min(low[u],\;low[v]) low[u]=min(low[u],low[v])

    • (u, v)(u,\,v)(u,v) 是指向栈中节点的一条横叉边，即 vvv 已被搜索且在栈中。这说明，从 vvv 开始搜索，经过一条路径到达 uuu ，此时又可以回到 vvv ，那这条路径上所有的点构成了一个环，即强连通分量。

      low[u]=min(low[u],  dfn[v])low[u] = min(low[u],\;dfn[v]) low[u]=min(low[u],dfn[v])

拓扑排序

这个简单。

记录每个点的入度，首先把入度为 000 的点入队。

随后每次从队首取出一个，把它指向的节点入度减一。如果此时入度被减为 000 ，那么入队。

记录下入队顺序，即是拓扑序列。

正确性显然：一个点的入度为 000 ，只能是所有指向它的点都为其减一，说明所有指向它的点都已入队并弹出，那么这个点就可以入队了，一定在指向它的所有点之后。

当然，这个容器不一定是队列。但选择队列，根据 FIFO 原则，就可以不用入队的时候记录顺序，而在出队的时候记录，更为方便。

/**********************************

Problem: 【模板】缩点 

State: Accepted

From: 【模板】 

Algorithm: tajan, topsort, DP

Author: cyh_toby

last updated on 2020.3.3

**********************************/
#include <cstdio>
#include <algorithm>
#include <vector>
#include <map>
#include <utility>
using namespace std;
#define pb push_back
#define pii pair<int,int>
const int N = 10005;
int n, m, num = 0;
int a[N];
vector<int> g1[N], g2[N];
map<pii, int> mp;
int dfn[N], low[N], book[N], id[N];
int f[N], deg[N], w[N];
int tot=0;
int s[N], tp = 0;
int q[N], hd, tl;
int ans = 0;
void tarjan(int cur)
{
	dfn[cur] = low[cur] = ++tot;
	s[++tp] = cur;
	book[cur] = 1;
	int len = g1[cur].size();
	for (int i = 0; i < len; i++) {
		int t = g1[cur][i];
		if (!dfn[t]) {
			tarjan(t);
			low[cur] = min(low[cur], low[t]);
			continue;
		}
		if (book[t])
			low[cur] = min(low[cur], dfn[t]);
	}
	if (dfn[cur] == low[cur]) {
		num++;
		int t;
		do {
			t = s[tp--];
			w[num] += a[t];
			id[t] = num;
			book[t] = 0;
		} while(t != cur);
	}
	return;
}
inline void make_graph()
{
	for (int i = 1; i <= n; i++) {
		int len = g1[i].size();
		int x = id[i], y;
		for (int j = 0; j < len; j++) {
			y = id[g1[i][j]];
			if (x==y) 
				continue;//自环 
			if (mp[make_pair(x,y)]==1)
				continue;//重边 
			mp[make_pair(x,y)] = 1;
			g2[x].pb(y);
			deg[y]++;
		}
	}
	return;
}
inline void top_sort_dp()//拓扑 + DP 
{
	hd = tl = 0;
	for (int i = 1; i <= num; i++)
		if (!deg[i])
			q[tl++] = i;
	while (hd != tl) {
		int x = q[hd++];
		f[x] += w[x];
		ans = max(ans, f[x]);
		int len = g2[x].size();
		for (int i = 0; i < len; i++) {
			int y = g2[x][i];
			f[y] = max(f[y], f[x]);
			if (--deg[y] == 0)
				q[tl++] = y;
		}
	}
	return;
}
int main()
{
	scanf("%d%d", &n, &m);
	for (int i = 1; i <= n; i++) 
		scanf("%d", &a[i]);
	for (int i = 1; i <= m; i++) {
		int x, y;
		scanf("%d%d", &x, &y);
		g1[x].pb(y);
	}
	for (int i = 1; i <= n; i++)
		if (!dfn[i])
			tarjan(i);
	make_graph();
	top_sort_dp();
	printf("%d", ans);
	return 0;
}
