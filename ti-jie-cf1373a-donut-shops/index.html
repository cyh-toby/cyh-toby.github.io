题解 CF1373A 【Donut Shops】

2020-06-26 17:29 2 min read # 题解

题意

原题：CF1373A【Donut Shops】

给定正整数 a, b, ca,\,b,\,ca,b,c ，两种方案：

  • 单个买，每个费用 aaa ；

  • 套装买 ，一个套装 bbb 个，每套费用 ccc ，可以买多但不能拆开。

求购买数量 xxx 或判定无解：

  1. 单个买费用严格小于套装买费用；

  2. 套装买费用严格小于单个买费用。

分析

称上述单个买的方案费用为 AAA，套装买费用为 BBB。

首先考虑什么时候 AAA 一定更小。
显然是 a<cba < \dfrac{c}{b}a<bc​ 。此时对于 111 任意一个 xxx 即可，并且 222 一定无解。

此外还有什么时候 AAA 可以更小？
注意到 BBB 要向上取整，又由于 2≤b2 \le b2≤b，因此当 a<ca<ca<c 时，x=1x=1x=1 即满足 111 。

除此以外 AAA 不可能更小，即 111 无解。

那么什么时候 BBB 可以更小？
显然想最优化 BBB，一定要当 b∣xb|xb∣x 时（这样相对于零售才不会有浪费）。这个时候用套装买其实就等价于单价 cb\dfrac{c}{b}bc​ 的零售。因此当 a×b>ca\times b > ca×b>c 时取 x=bx = bx=b 即满足 222。反之则 222 无解。

将上述整理写出代码

源码

#include <bits/stdc++.h>

using namespace std;

typedef long long ll;

int main()
{
	int T;
	scanf("%d", &T);
	while (T--) {
		ll a, b, c;
		scanf("%lld%lld%lld", &a, &b, &c);
		if (a < (double)c / b) {
			printf("1 -1\n");
			continue;
		}
		if (a < c) {
			printf("1 ");
		}
		else {
			printf("-1 ");
		}
		if (a*b > c) {
			printf("%lld\n", b);
		}
		else
			printf("-1\n");
	}
	return 0;
}
