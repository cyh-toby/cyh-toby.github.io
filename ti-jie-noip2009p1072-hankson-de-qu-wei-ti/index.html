题解 [NOIp2009]/P1072 【Hankson的趣味题】

2020-03-15 14:47 2 min read # 题解

题意

原题：P1072 Hankson 的趣味题

给定 a,b,c,da,b,c,da,b,c,d 四个正整数，求使下条件同时成立的正整数 xxx 的个数。

  • gcd⁡(x,a)=b\gcd(x,a)=bgcd(x,a)=b

  • lcm(x,c)=d\text{lcm}(x,c)=dlcm(x,c)=d

分析

第一个条件不好入手找 xxx ，考虑从第二个条件攻破。

显然 xxx 必须是 ddd 的因数，而找出一个数的因数的复杂度是 O(n)O(\sqrt{n})O(n​) ，故可以考虑枚举 xxx 。

枚举 xxx 判断是否满足条件即可。

一次 gcd⁡\gcdgcd 复杂度 log⁡ n\log\,nlogn 。

总复杂度 O(Tnlog⁡n)O(Tn\log n)O(Tnlogn) ，nnn 为值域最大值。

勉强可以过。

源码

/**********************************

Problem: [NOIp2009TG] - Hankson的趣味题 

State: Accepted

From: [NOIp2009TG]

Algorithm: 数论, gcd 

Author: cyh_toby

Last updated on 2020.3.15

**********************************/
#include <cstdio>
using namespace std;
const int N = 50005;
int a, b, c, d, n, div[N], cnt, ans;
int gcd(int x, int y) {
	if (y == 0)
		return x;
	return gcd(y, x % y);
}
int main()
{
	scanf("%d", &n);
	while (n--) {
		scanf("%d%d%d%d", &a, &b, &c, &d);
		int t = d;
		ans = cnt = 0;
		for (int i = 1; i <= t/i; i++) {
			if (t % i == 0) {
				div[++cnt] = i;
				if (i * i != t)
					div[++cnt] = t/i;
			}
		}
		for (int i = 1; i <= cnt; i++) {
			int e = div[i];
			if(gcd(e, a) == b && e*c == d*gcd(e,c))
				ans++; 
		}
		printf("%d\n", ans);
	}
	return 0;
}
