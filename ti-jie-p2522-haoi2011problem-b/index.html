题解 P2522 【[HAOI2011]Problem b】

题意

原题：P2522 【[HAOI2011]Problem b】

  • 求 ∑i=ab∑j=cd[gcd⁡(i,j)=k]\sum\limits_{i=a}^b\sum\limits_{j=c}^d\left[\gcd(i,j)=k\right]i=a∑b​j=c∑d​[gcd(i,j)=k]。

  • 多组数据。

分析

首先可以把问题转化为求 ∑i=1n∑j=1m[gcd⁡(i,j)=k]\sum\limits_{i=1}^n\sum\limits_{j=1}^m\left[\gcd(i,j)=k\right]i=1∑n​j=1∑m​[gcd(i,j)=k] 。

显然这个等价于求 ∑i=1⌊n/k⌋∑j=1⌊m/k⌋[gcd⁡(i,j)=1]\sum\limits_{i=1}^{\lfloor n/k\rfloor}\sum\limits_{j=1}^{\lfloor m/k\rfloor}\left[\gcd(i,j)=1\right]i=1∑⌊n/k⌋​j=1∑⌊m/k⌋​[gcd(i,j)=1]，把 n,mn,mn,m 都除以 kkk 下取整，那么转化为求 ∑i=1n∑j=1m[gcd⁡(i,j)=1]\sum\limits_{i=1}^n\sum\limits_{j=1}^m\left[\gcd(i,j)=1\right]i=1∑n​j=1∑m​[gcd(i,j)=1]。

接下来证明上式等价于 ∑d=1min⁡(n,m)μ(d)⌊nd⌋⌊md⌋\sum\limits_{d=1}^{\min(n,m)}\mu(d) \lfloor\frac{n}{d}\rfloor\lfloor\frac{m}{d}\rfloord=1∑min(n,m)​μ(d)⌊dn​⌋⌊dm​⌋

法一

由莫比乌斯函数性质，即 μ∗1=ϵ\mu * 1=\epsilonμ∗1=ϵ 展开可知

∑d∣gcd⁡(i,j)μ(d)=ϵ(gcd⁡(i,j))=[gcd⁡(i,j)=1]\sum\limits_{d|\gcd(i,j)}\mu(d)=\epsilon(\gcd(i,j))=[\gcd(i,j)=1]d∣gcd(i,j)∑​μ(d)=ϵ(gcd(i,j))=[gcd(i,j)=1]

所以

原式=∑i=1n∑j=1m∑d∣gcd⁡(i,j)μ(d)=∑i=1n∑j=1m∑d∣i,d∣jμ(d)=∑d=1min⁡(n,m)μ(d)(∑d∣in∑d∣jn1)=∑d=1min⁡(n,m)μ(d)⌊nd⌋⌊md⌋\begin{aligned}\text{原式} & =\sum\limits_{i=1}^n\sum\limits_{j=1}^m\sum\limits_{d|\gcd(i,j)}\mu(d) \\ & = \sum\limits_{i=1}^n\sum\limits_{j=1}^m\sum\limits_{d|i, d|j}\mu(d) \\ & = \sum\limits_{d=1}^{\min(n,m)}\mu(d)\left(\sum\limits_{d|i}^n\sum\limits_{d|j}^n 1\right) \\ & = \sum\limits_{d=1}^{\min(n,m)}\mu(d) \lfloor\frac{n}{d}\rfloor\lfloor\frac{m}{d}\rfloor\end{aligned}原式​=i=1∑n​j=1∑m​d∣gcd(i,j)∑​μ(d)=i=1∑n​j=1∑m​d∣i,d∣j∑​μ(d)=d=1∑min(n,m)​μ(d)⎝⎛​d∣i∑n​d∣j∑n​1⎠⎞​=d=1∑min(n,m)​μ(d)⌊dn​⌋⌊dm​⌋​

法二

容斥。

记 D(a,b,k)D(a,b,k)D(a,b,k) 表示有多少 x≤a,y≤bx\le a, y \le bx≤a,y≤b 满足 k∣x,k∣yk|x,k|yk∣x,k∣y，则 D(a,b,k)=⌊ak⌋⌊bk⌋D(a,b,k) = \lfloor\frac{a}{k}\rfloor\lfloor\frac{b}{k}\rfloorD(a,b,k)=⌊ka​⌋⌊kb​⌋ 。

从容斥角度思考，用总的减去不合法的，补回多减去的，再继续减去......可以得到原式即为 ∑d=1min⁡(n,m)μ(d)D(n,m,d)=∑d=1min⁡(n,m)μ(d)⌊nd⌋⌊md⌋\sum\limits_{d=1}^{\min(n,m)}\mu(d)D(n, m, d)=\sum\limits_{d=1}^{\min(n,m)}\mu(d) \lfloor\frac{n}{d}\rfloor\lfloor\frac{m}{d}\rfloord=1∑min(n,m)​μ(d)D(n,m,d)=d=1∑min(n,m)​μ(d)⌊dn​⌋⌊dm​⌋ 。

可参考《算法竞赛进阶指南》0x37 节例题 【Zap】。

上述两种方法证明后，只需要先预处理 μ\muμ 的前缀和，再用数论分块处理两个下取整部分即可。单次的时间复杂度可以做到 O(n+m)O(\sqrt n + \sqrt m)O(n​+m​)。

源码

const int N = 5e4+5;
typedef long long ll;

int p[N], vis[N], cnt, miu[N], s[N];

inline void init() {
	miu[1] = 1;
	for (int i = 2; i < N; i++) {
		if (!vis[i]) p[++cnt] = i, miu[i] = -1;
		for (int j = 1; j <= cnt && i * p[j] < N; j++) {
			vis[i * p[j]] = 1;
			if (i % p[j] == 0) {
				miu[i * p[j]] = 0;
				break;
			}
			miu[i * p[j]] = -miu[i];
		}
	}
	for (int i = 1; i < N; i++) s[i] = s[i-1] + miu[i];
}

inline ll calc(int n, int m) {
	ll ans = 0;
	for (int l = 1, r; l <= min(n, m); l = r + 1) {
		r = min(n/(n/l), m/(m/l));
		ans += 1ll * (n/l) * (m/l) * (s[r] - s[l-1]);
	}
	return ans;
}

int main() 
{
	init();
	int T;
	scanf("%d", &T);
	while (T--) {
		int a, b, c, d, k;
		scanf("%d%d%d%d%d", &a, &b, &c, &d, &k);
		printf("%lld\n", calc(b/k, d/k) - calc((a-1)/k, d/k) - calc(b/k, (c-1)/k) + calc((a-1)/k, (c-1)/k));
	}
	return 0;
}
