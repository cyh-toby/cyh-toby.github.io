题解 CF1373C 【Pluses and Minuses】

2020-06-26 20:56 2 min read # 模拟 # 题解

题意

原题：CF1373C 【Pluses and Minuses】

分析

把题目给的伪代码稍微转化一下，就是说，对于给定的 010101 序列 aaa 的前缀和序列 sss ，从前往后遍历，一旦遇到一个位置 iii ，使得 si<0s_i < 0si​<0 ，就把整个 sss 序列通通加上 111 （即把这个小于 000 的地方补上去，使之大于等于 000 ；而由于是 010101 串的前缀和，只需要加 111 达成目的），然后答案 resresres 加上 iii 。

其中”把整个序列通通加上 111“ 可以用题目中 curcurcur 的方式处理。

将上述转化为代码即可，算法时间复杂度线性。

源码

#include <bits/stdc++.h>

using namespace std;

const int N = 1e6+5;
typedef long long ll;

int n;
char s[N];
ll ans;

int main()
{
	int T;
	scanf("%d", &T);
	while (T--) {
		ans = 0;
		
		scanf("%s", s+1);
		n = strlen(s+1);
		int sum = 0, cur = 0;
		for (int i = 1; i <= n; i++) {
			if (s[i] == '+')
				sum++;
			else
				sum--;
			if (sum + cur < 0) {
				ans += i;
				cur++;
			}
		}
		
		printf("%lld\n", ans + n);
	}
	
	return 0;
}
